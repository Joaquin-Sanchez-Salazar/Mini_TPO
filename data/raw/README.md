# Datos raw

Esta carpeta no versiona los datos de la prueba técnica ni su PDF porque no existe autorización explícita para publicarlos.

Para ejecutar el pipeline, coloque aquí:

- `base_mini_tpo.csv`: histórico promocional en formato CSV.
- `prueba_tecnica_ds.pdf`: documento de contexto de la prueba.

El esquema requerido del CSV se define en `configs/project_config.yaml`. El pipeline usa rutas relativas y no modifica estos archivos.

No publique datos reales, credenciales ni documentos de terceros sin autorización.
